How The Agent Logic Works

  1. Inbound Webhook: Receives role title, JD text, and candidate resume list.
  2. Score + Sort Candidates: Deterministic scoring using fixed weights: Skills 40, Experience 25, Tools 20, Soft Skills 15.
  3. Recommendation Rules: Interview if score >= 75, Hold if 50-74, Reject if < 50.
  4. Output: Ranked candidates + grouped buckets (Interview, Hold, Reject).
  5. Respond Node: Returns structured JSON to frontend/ATS.
